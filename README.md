# 3awan Caferesto App

Aplikasi mobile Flutter untuk manajemen cafe restaurant (caferesto) yang dikembangkan sebagai tugas mata kuliah Mobile Programming.

## Deskripsi Proyek

Aplikasi ini adalah aplikasi mobile yang dibangun menggunakan Flutter untuk mengelola operasional sebuah cafe restaurant. Aplikasi ini menyediakan berbagai fitur untuk memudahkan pengelolaan menu, pesanan, dan transaksi di cafe restaurant.

## Teknologi yang Digunakan

- **Flutter** - Framework untuk pengembangan aplikasi mobile cross-platform
- **Dart** - Bahasa pemrograman yang digunakan dalam Flutter

## Fitur Aplikasi

- Manajemen menu makanan dan minuman
- Sistem pemesanan
- Manajemen transaksi
- Interface yang user-friendly

## Getting Started

### Prerequisites

- Flutter SDK (versi terbaru)
- Dart SDK
- Android Studio / VS Code dengan Flutter extension
- Emulator atau device fisik untuk testing

### Installation

1. Clone repository ini:
```bash
git clone [url-repository]
```

2. Masuk ke direktori proyek:
```bash
cd 3awan_caferesto_app
```

3. Install dependencies:
```bash
flutter pub get
```

4. Jalankan aplikasi:
```bash
flutter run
```
